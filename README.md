# The Sports Bar Co.

A multi-page website for a fictional game-day sports bar, built from scratch
with vanilla HTML, CSS, and JavaScript. Part of a front-end practice series
alongside [the-salon-co](#) and [the-bakery-co](#).

## About

The Sports Bar Co. is a classic American sports bar concept — warm, dark,
neon-lit "game-day cave" energy. This project recreates the full marketing
site: menu, live game schedule, events, reservations, and more.

Built as a hands-on exercise in semantic HTML, modern CSS layout
(flexbox + grid), CSS custom properties, and DOM-based interactivity.

## Pages

- **Home** — brand hero, tonight's games, weekly specials, floor plan
- **Menu** — full food & drink menu with category filters
- **Games** — live game schedule with day/sport filters and a TV map
- **Events** — events calendar, daily specials, and happy hour
- **Reserve** — table booking, private events, and online ordering
- **About** — story, timeline, the crew, and job openings
- **Contact** — location, hours, contact form, and socials

## Built With

- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties (design tokens), flexbox, grid, media queries
- **JavaScript** — vanilla DOM interactivity (filters, tabs, forms)
- No frameworks, no build step.

## Design System

- **Type:** Big Shoulders Display (headlines), DM Sans (body), JetBrains Mono (data)
- **Palette:** warm dark backgrounds, cream text, neon amber / red / cyan accents
- **Motifs:** scoreboard panels, neon signage, leather & wood textures
